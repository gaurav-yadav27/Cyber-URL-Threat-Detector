## 📌 Overview

Cyber URL Threat Detector is a professional-grade cybersecurity tool that analyzes URLs for malicious indicators, phishing attempts, and cyber threats.

### ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 Single URL Scan | Real-time threat analysis with visual feedback |
| 📊 Batch Analysis | Scan multiple URLs from a text file |
| 📜 Scan History | SQLite database stores all previous scans |
| 📈 Statistics Dashboard | View threat distribution and trends |
| 🔑 API Integration | Optional VirusTotal API for enhanced detection |
